# Lesson 08 - Capstone Project

## Project options

Choose one:

1. synchronous FIFO
2. round-robin arbiter
3. programmable counter controller

## Required deliverables

- module specification
- interface table
- behavior rules
- corner-case section
- test plan matrix
- at least 8 directed tests
- RTL implementation
- BDD scenario list
- reflection on design tradeoffs

## Evaluation rubric

- clarity of spec
- completeness of edge cases
- traceability from spec to tests
- RTL readability
- behavior protection through tests and assertions

## Recommended workflow

1. write spec
2. review spec
3. build test plan
4. implement TDD cycle
5. add BDD scenario sheet
6. run regression and summarize results
