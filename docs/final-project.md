# Final Project

## Objective

Apply SDD, TDD, and BDD to a realistic RTL block and produce a professional-grade design package.

## Suggested project topics

- synchronous FIFO
- round-robin arbiter
- programmable watchdog / timer controller
- request / grant controller

## Final package must include

- problem statement
- formalized specification
- interface contract
- corner cases
- test plan matrix
- BDD scenarios
- RTL implementation
- testbench strategy
- assertion candidates
- regression summary
- lessons learned

## What good looks like

A strong final project demonstrates disciplined engineering flow, not just working code.
