# Quiz 02

## Multiple choice

1. In TDD, what should come first?
   - A. optimized RTL
   - B. a failing test
   - C. waveform screenshot
   - D. lint cleanup

2. Which test is a boundary test for a saturating counter?
   - A. reset to zero
   - B. count from 10 to 11
   - C. count from max-1 to max
   - D. disable enable signal forever

## Short answer

3. Why should tests focus on behavior instead of internal implementation details?
4. What does regression protect during refactoring?

## Answer key

1. B
2. C
3. Because implementation can change while required behavior remains the same.
4. It protects expected external behavior from accidental breakage.
