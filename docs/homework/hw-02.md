# Homework 02 - Build a Test Plan from a FIFO Spec

## Assignment

Assume you are given a synchronous FIFO spec. Create a test plan matrix.

## Minimum required tests

- reset behavior
- first write
- first read
- full condition
- empty condition
- simultaneous read/write
- overflow attempt
- underflow attempt

## Deliverable format

| Requirement ID | Description | Test Case Name | Type |
|---|---|---|---|

Type should be one of:
- positive
- boundary
- negative

## Stretch goal

Add a coverage note for each test describing what behavior it protects.
