# Homework 05 - Mini Capstone Review

## Assignment

Choose one small RTL block:
- FIFO
- arbiter
- counter controller

Create a mini capstone package containing:

1. 1-page specification
2. 1 test matrix
3. 4 BDD scenarios
4. 3 assertion candidates
5. 1 short reflection paragraph

## Goal

This homework checks whether you can connect the whole course flow into one coherent engineering package.

## Review rubric

- clarity
- completeness
- traceability
- engineering realism
