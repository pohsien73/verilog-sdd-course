# Homework 03 - Write BDD Scenarios for a Round-Robin Arbiter

## Assignment

Write at least 6 Given / When / Then scenarios for a 4-request round-robin arbiter.

## Required scenario categories

- idle state
- single requester
- multiple simultaneous requesters
- fairness across cycles
- reset behavior
- requester drops request mid-sequence

## Goal

Each scenario should be detailed enough that another engineer could convert it into a testcase without asking you for clarification.

## Stretch goal

For 2 scenarios, also describe what assertion you would want in simulation.
